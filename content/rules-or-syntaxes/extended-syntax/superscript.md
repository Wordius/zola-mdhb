+++
title = "Superscript in markdown"
description = "Superscript is a character set, printed or written above and immediately to one side of another"
weight = 1000
path = "superscript-in-markdown"
+++

Use a caret (`^`) before and after the characters:

```markdown
X^2^
```

Which renders as:

X<sup>2</sup>

Alternatively, use HTML:

```html
X<sup>2</sup>
```
